# KOOK 群管理插件

AstrBot 插件 - KOOK 平台群管理工具

## 功能

- 新成员加入时自动发送欢迎卡片
- 支持自定义欢迎语、按钮链接
- 支持配置目标频道
- 更多功能开发中...

## 安装

在 AstrBot WebUI 中, 选择 "从链接安装", 输入本仓库地址。

## 配置

安装后在插件配置中设置:

| 配置项 | 说明 |
|--------|------|
| welcome_channel_id | 欢迎消息发送的目标频道ID |
| server_name | 服务器名称 |
| button1_link | 第一个按钮跳转链接 |
| button2_link | 第二个按钮跳转链接 |

## 获取频道ID

在 KOOK 客户端中, 右键点击频道 -> 复制ID
